# Sports Facility Court Booking Platform

A full-stack web application for booking sports courts.

## Features
- **Frontend**: React.js with a premium, colorful UI.
- **Backend**: Node.js/Express with MongoDB.
- **Booking**: Interactive court selection and scheduling.

## Getting Started

### Backend
1. Navigate to `backend`.
2. Run `npm install`.
3. Start server: `node server.js` (Runs on port 5001).

### Frontend
1. Navigate to `frontend-app`.
2. Run `npm install`.
3. Start application: `npm start` (Runs on port 3000).

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
